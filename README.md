Student Course Hub
A simple web application where students can view courses, register interest, and see modules with lecturers.
________________________________________
Features
                   Student
•	Student login and signup
•	View available programmes
•	Register interest in courses
•	Withdraw interest
•	View modules and lecturers
                Staff
•	View modules they lead
•	Update their profile
•	Staff login
•	View the programmes which have modules they teach
•	Interested Students list in programmes that contain the modules they teach

Setup
1.	Install XAMPP and start Apache & MySQL
2.	Move project folder to:
3.	C:\xampp\htdocs\
4.	Create a database in phpMyAdmin
5.	Update config.php with your database details
6.	Open in browser:
7.	http://localhost/student_course_hub/
